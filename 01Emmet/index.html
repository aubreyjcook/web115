Emmet is an IDE plugin

Emmet is useful for someone like me, who writes HTML and CSS in an IDE (Integrated development environment) to create webpages. Emmet shortens the number of keystrokes needed to write a line of code in a document, so long as you know the right keywords to use. Emmet documentation can be found at Emmet.io

Responsive Web Design

Another important aspect of this site is that it is responsive, this means it checks to see what type of device it is being accessed on by the user, and the relative screen size of the display. Once this information is determined it formats the page to fit the size correctly, without this you wouldn't be able to view this site on mobile devices! This is accomplished by using a CSS boilerplate that uses a grid system, called 'Skeleton.' Information about skeleton and how to use it is available at getskeleton.com
